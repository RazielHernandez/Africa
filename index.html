I'mAfrica Gallery App

Developed by Raziel Hernandez

This App was designed following the course "SwiftUI + SwiftData Masterclass - iOS App Development Swift" and implements some of the basic and intermediate principles of SwiftUI such as creation of views, logo, colorsets, home screen, image gallery, implementation of maps with custom bookmarks, video playback and JSON file handling

Stylized design with a thumbnail list of the different animals
Image gallery with button to change the view, varying the number of columns
View with details of each animal, including two different galleries (images and facts)
Video list that allows the user to automatically play a video when selecting it
Gallery with dynamic number of columns, controlled by a slider
Implementation of a map with custom markers
